eINFO API Interface with Canadian post-secondary data.

This is an open source project. Contribute here.

Usage

Access the API endpoint with the eINFO program code encoded as query parameter c.

/api?c={code}

Find the code parameter through the eINFO URL, i.e. http://www.electronicinfo.ca/programs/203/.
